This repository contains all the data, scripts, and code for the paper titled "A Closer Look at the Use of Reinforcement Learning for Speeding Up Runtime Verification of Software Tests," submitted to ISSTA 2026.

- `data/`: Statistics of evaluation subjects, online/offline-tuned hyperparameters, and time series and trajectories of all projects 
- `code/`: Implementations of time series collection, the theoretically optimal baseline, and different RL algorithms (D-UCB and DS-TS) 
- `rq{1,2,3,4,5}/`: Data, scripts, and code used for each RQ
